    Setting up virtual env using python

    Python 3.3+ comes with venv built-in ‚Äî no need to install anything else.

    üöÄ Step-by-Step Instructions¶

    üîπ Step 1: Create a Virtual Environment¶

    Navigate to your project folder:

    cd path/to/your/project

    Then run:

    python -m venv venv

    This creates a folder called venv/ with a self-contained Python environment.

    üîπ Step 2: Activate the Environment¶

    ü™ü On Windows:¶

    venv\Scripts\activate

    üêß On Linux/macOS:¶

    source venv/bin/activate

    ‚úÖ You'll know it's activated if your terminal prompt starts with:

    (venv)

    üîπ Step 3: Install Dependencies¶

    Once activated, install your packages using pip:

    pip install numpy pandas matplotlib

    üîπ Step 4: Freeze Your Environment (Optional)¶

    Save current packages to a file:

    pip freeze > requirements.txt

    To recreate the environment later:

    pip install -r requirements.txt

    üîπ Step 5: Deactivate the Environment¶

    When you're done:

    deactivate

    üß† Bonus Tips¶

    Task Command
    Delete environment Just delete the venv/ folder

    Use environment in VSCode It will auto-detect or use Python: Select Interpreter
